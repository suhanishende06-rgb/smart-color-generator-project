# Smart Color Generator

Smart Color Generator is a simple and interactive JavaScript project that generates random colors.  
It allows users to switch between HEX and RGB formats, generate gradients, and copy color codes easily.

## Features
- Random color generation
- HEX and RGB format support
- Gradient color generator
- Copy color code with one click
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Files Used
- index.html
- style.css
- script.js

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Click buttons to generate and copy colors
